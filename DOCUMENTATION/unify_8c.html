term term_unify ( term  t)

The term system is given in the form 

unify ( = ( <t1> <t1'> ) … = ( <tn> <tn'> ) )

The initial term is not modified. A new term is generated with the form 

solution ( val ( <variable> <value> ) … val ( <variable> <value> ) )

If no solution is possible, then the output is of the form 

incompatible ( <term> <term> )
Parameters
tencode the terms to unify.
Precondition
t is not NULL and should be of the correct form.
Returns
solution term or impossible term.
