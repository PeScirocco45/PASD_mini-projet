Typedef Documentation

typedef struct term_list_struct * term_list

This structure is used to have a both way linked list of terms. It is used to record the arguments of terms. It is linked in both way to ensure simple navigation.

typedef struct term_struct term_struct

This structure is used to record a term. All sub-term / arguments are stored in a both way linked list of terms. Arguments can be accessed from both side.
