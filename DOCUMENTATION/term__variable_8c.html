bool term_is_variable ( term  t)

To test whether a term is a variable. A variable has an symbol of the following form:

  • first char is '\''
  • the next symbol belongs to 'a'-'z' 'A'-'Z' '_'
  • every other char in 'a'-'z' 'A'-'Z' '0'-'9' '_' A variable should not have any argument.

No side effect, can be used in assert.

Postcondition
it is asserted that if it is a variable, then the arity is 0
Returns
true if the term is a variable.
